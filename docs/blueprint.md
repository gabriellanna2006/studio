# **App Name**: RaulFind

## Core Features:

- Animal Registration: Allow users to register lost or found animals with details like species, breed, color, size, and location found in Raul Soares, MG.
- Owner Registration: Enable pet owners to register their contact information (address, phone number, etc.) in case their pet is found.
- Unique Animal Link Generation: Generate a unique, short URL (e.g., raulfind.com/abcd) for each registered animal that can be easily shared and accessed on mobile devices.
- Animal Profile Display: Display the animal's information and the owner's contact details on a dedicated page when someone accesses the animal's unique link.
- Search Functionality: Implement a search feature for users to search for lost or found animals based on different criteria.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of trust and tranquility.
- Background color: Very light blue (#F0F8FF) for a clean and airy feel.
- Accent color: Light orange (#FFB347) to highlight important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif font that's versatile enough for both headlines and body.
- Use simple, clear icons related to pets and location (e.g., paw prints, map markers).
- Clean and responsive layout, optimized for mobile devices, with easy-to-find information.
- Subtle animations for loading states and user interactions to provide a smooth experience.
